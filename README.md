# Bitasmbl-AI-Powered-Study-Notes-Generator-82c99b

## Description
Build a web application that allows students to input lectures, textbooks, or articles and automatically generate concise, structured study notes using AI summarization and keyword extraction while maintaining readability and context.

## Tech Stack
- Python
- Vue.js
- Tailwind CSS

## Requirements
- Python
- Vue.js
- Tailwind CSS

## Installation
bash
git clone https://github.com/MrBitasmblTester2/Bitasmbl-AI-Powered-Study-Notes-Generator-82c99b.git
cd Bitasmbl-AI-Powered-Study-Notes-Generator-82c99b


## Usage
- Run Python backend
- Run Vue.js frontend with Tailwind CSS

## Implementation Steps
1. Set up Python backend for processing text input.
2. Integrate AI summarization and keyword extraction in Python.
3. Create Vue.js frontend for input and displaying notes.
4. Style views using Tailwind CSS.
5. Connect frontend to backend endpoints.
